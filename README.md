## 1. Installation
Erstelle eine virtuelle Umgebung:
```powershell
python -m venv .venv
```
Aktiviere die virtuelle Umgebung (in der Windows-Eingabeaufforderung):
```powershell
.venv\Scripts\activate.bat
```
Installiere alle benötigten Abhängigkeiten:
```powershell
pip install -r requirements.txt
```

## 2. Ausführung
Starte das Programm:
```powershell
python main.py
```
